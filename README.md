## 3D图形术语
  * Transformation: 变换
  * Transformation Matrix: 变换矩阵
  * Projection: 投影
  * Projection Matrix: 投影矩阵
  * Vertex: 顶点
  * Rasterization: 光栅化，实际绘制或者填充每个顶点之间的像素形成线段就叫光栅化
  * Shading: 着色
  * Texture Mapping: 文理贴图
  * Blending: 混合
  * 2D笛卡尔坐标
  * 3D笛卡尔坐标
  * 视口: 坐标系统从逻辑笛卡尔坐标映射到物理屏幕像素坐标
